# Truck and Load API

This project is a NodeJS, Express, MongoDB, and Mongoose API for managing trucks and loads.

## Collections
- Truck
- Load

## Relationship
A load has a truck.

## Features
- Full CRUD for trucks
- Full CRUD for loads
- Mongoose schemas and validation
- Error handling with proper HTTP status codes
- Separate routes and controllers

## Technologies Used
- Node.js
- Express
- MongoDB
- Mongoose
- Dotenv
- Nodemon
